# 섹션

이 파일은 모든 섹션, 순서, 영향도 수준 및 설명을 정의합니다.
괄호 안의 섹션 ID는 규칙을 그룹화하는 데 사용되는 파일명 접두사입니다.

---

## 1. 컴포넌트 아키텍처 (architecture)

**영향도:** HIGH
**설명:** prop 증식을 방지하고 유연한 합성을 가능하게 하는 컴포넌트 구조화의 기본 패턴.

## 2. 상태 관리 (state)

**영향도:** MEDIUM
**설명:** 상태를 끌어올리고 합성된 컴포넌트 간에 공유 컨텍스트를 관리하는 패턴.

## 3. 구현 패턴 (patterns)

**영향도:** MEDIUM
**설명:** 복합 컴포넌트와 컨텍스트 프로바이더를 구현하는 구체적인 기법.

## 4. React 19 API (react19)

**영향도:** MEDIUM
**설명:** React 19 이상 전용. `forwardRef`를 사용하지 말고, `useContext()` 대신 `use()`를 사용할 것.
